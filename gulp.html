gulp-scss-starter
Install #Gulp Команды #Gulp Еще что то
  • установите NodeJS
  • установите глобально:
  • Yarn: npm i -g yarn
  • Gulp: npm i -g gulp
  • Bem Tools: npm i -g bem-tools-core
  • -- "git clone https://github.com/andreyalexeich/gulp-scss-starter.git" - Git Clone

  • перейдите в скачанную папку со сборкой: cd gulp-scss-starter
  • введите yarn set version berry
  • скачайте необходимые зависимости: yarn
  • -- "yarn run dev" - Режим разработки
  • -- "yarn run build" - Режим сборки
