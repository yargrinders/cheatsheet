git commit

Комманда git commit- используется для записи индексированных изменений в репозиторий. Используйте эту метку для обозначения всех вопросов, связанных с созданием, редактированием и внутренней структурой коммитов в Git.

git branch

Команда git branch- это своего рода "менеджер веток". Она умеет перечислять ваши ветки, создавать новые, удалять и переименовывать их.

git checkout

Команда git checkout- используется для переключения веток и выгрузки их содержимого в рабочий каталог.

git marge

Команда git marge- используется для слияния рабочих веток имя-ветки2 в имя-ветки(main)
