#Git commit #Git Branch #Git checkout #Git marge
  • -- "git add ." -Добавить всё
  • -- "git commit -m "versinon"- Комментарий
  • -- "git push"- Загрузить в Git
  • -- "git branch" - Простой список имеющихся у вас веток:
  • -- "git branch имя-ветки" - Создать имя-ветки
  • -- "git branch -d имя-ветки" - Удаление ветки
  • -- "git checkout" - переключения веток и выгрузки их содержимого в рабочий каталог.

  • -- "git checkout -b имя-ветки"- шорткат и переключиться на имя-ветки
  • -- "git marge имя-ветки" - git checkout master в git merge style

git branch

Команда git branch — это своего рода "менеджер веток". Она умеет перечислять ваши ветки, создавать новые, удалять и переименовывать их.

git checkout

Команда git checkout используется для переключения веток и выгрузки их содержимого в рабочий каталог.

git marge

Команда git margeиспользуется для слияния рабочих веток имя-ветки2 в имя-ветки
